Height Converter

A simple human height converter between feet, inches and centimeters.

How to use

Just enter the height in the corresponding fields and the conversion will be done automatically.

  • This converter allows for imperial-to-metric and metric-to-imperial conversions.
  • Use the decimal drop-down to select a decimal precision for the heights.

Why a height converter?

This is an effort to help all people who, like me, have a hard time imagining a person's height using another measurement system.
